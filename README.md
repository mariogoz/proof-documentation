# Prueba Walmart

## Ejecucion

Descargar los repositorios

Front
```bash
git clone https://github.com/mariogoz/front-walmart.git
```

Back
```bash
git clone https://github.com/mariogoz/back-walmart.git
```

Ejecutar la instalacion de dependencias 
```bash
npm i
```

### Local

Proyecto Front ejecutar
```bash
npm run local
```

Proyecto Front ejecutar
```bash
node index.js
```

### Docker compose

```bash
docker-compose up
```
